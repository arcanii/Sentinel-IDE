SQLTerminal-Win32
-----------------
Sentinel-IDE's Win32 shell derives from SQLTerminal-Win32 (same author), which is
licensed GPL-3.0. The dark/coral palette in src/host/win32/Theme.h, the themed
dialog conventions, the tree/splitter handling, and the worker-thread build model
all originate there. This is why Sentinel-IDE is GPL-3.0-or-later rather than a
more permissive licence.


Appcast format
--------------
The auto-update feed follows the Sparkle appcast XML namespace
(http://www.andymatuschak.org/xml-namespaces/sparkle), originally from the Sparkle
project for macOS. The format is used here for interoperability with WinSparkle; no
Sparkle code is included in this repository.
